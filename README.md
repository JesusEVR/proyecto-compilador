# Proyecto de Compiladores

En la carpeta ***compilador-c*** se encuentra un compilador desarrollado en C para la generación de código intermedio en 3 direcciones, a partir de un lenguaje de programación propuesto con sintaxis inspirada en Java y C, pero en español.

En la carpeta ***compilador-java*** se encuentra un compilador desarrollado en Java. Se implementa un `analizador léxico` haciendo uso de autómatas finitos deterministas minimizados, así como tres `analizadores sintácticos`: LL(1), LR(1) y LALR(1).
